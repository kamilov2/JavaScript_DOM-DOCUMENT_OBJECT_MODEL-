DOM bilan ishlash bo'yicha qo'llanma

DOM nima?

DOM hujjat obyekt modeli bo‘lib, HTML yoki XML hujjatni obyektlar daraxti sifatida tasvirlaydi. HTML hujjatidagi har bir element JavaScript orqali foydalanish va o‘zgartirish mumkin bo‘lgan ob'ektga aylanadi.

DOM elementlariga kirish

1. document.getElementById(id)

Berilgan id ga ega elementni qaytaradi.

        
let element = document.getElementById('myId');
console.log(element);

2. document.getElementsByClassName(className)

Berilgan sinf nomiga ega elementlar to‘plamini qaytaradi.

        
let elements = document.getElementsByClassName('myClass');
console.log(elements);

3. document.getElementsByTagName(tagName)

Berilgan teg nomiga ega elementlar to‘plamini qaytaradi.

        
let elements = document.getElementsByTagName('div');
console.log(elements);

4. document.querySelector(selector)

CSS selektoriga mos keladigan birinchi elementni qaytaradi.

        
let element = document.querySelector('.myClass');
console.log(element);

5. document.querySelectorAll(selector)

CSS selektoriga mos keladigan barcha elementlarni qaytaradi.

        
let elements = document.querySelectorAll('.myClass');
console.log(elements);

Elementlarning mazmuni va atributlarini o‘zgartirish

1. element.innerHTML

Elementning HTML mazmunini oladi yoki o‘rnatadi.

        
let element = document.getElementById('myId');
element.innerHTML = '<p>Yangi mazmun</p>';

2. element.textContent

Elementning matn mazmunini oladi yoki o‘rnatadi.

        
let element = document.getElementById('myId');
element.textContent = 'Yangi matn mazmuni';

3. element.setAttribute(name, value)

Elementning atribut qiymatini o‘rnatadi.

        
let element = document.getElementById('myId');
element.setAttribute('class', 'newClass');

4. element.getAttribute(name)

Elementning atribut qiymatini oladi.

        
let value = document.getElementById('myId').getAttribute('class');
console.log(value);

5. element.removeAttribute(name)

Elementning atributini o‘chiradi.

        
let element = document.getElementById('myId');
element.removeAttribute('class');

Elementlarning uslubini o‘zgartirish

element.style.property

Elementning CSS xususiyatlarini o‘zgartiradi.

        
let element = document.getElementById('myId');
element.style.color = 'red';
element.style.backgroundColor = 'blue';

Klasslar bilan ishlash

1. element.classList.add(className)

Elementga klass qo‘shadi.

        
let element = document.getElementById('myId');
element.classList.add('newClass');

2. element.classList.remove(className)

Elementdan klassni olib tashlaydi.

        
let element = document.getElementById('myId');
element.classList.remove('oldClass');

3. element.classList.toggle(className)

Elementning klassini yoqadi yoki o‘chiradi.

        
let element = document.getElementById('myId');
element.classList.toggle('toggleClass');

4. element.classList.contains(className)

Elementda berilgan klass borligini tekshiradi.

        
let element = document.getElementById('myId');
let hasClass = element.classList.contains('myClass');
console.log(hasClass);

DOMga element qo‘shish va olib tashlash

1. document.createElement(tagName)

Yangi element yaratadi.

        
let newElement = document.createElement('div');

2. parentNode.appendChild(newChild)

Yangi elementni ota elementga qo‘shadi.

        
let parent = document.getElementById('parentId');
let newElement = document.createElement('div');
parent.appendChild(newElement);

3. parentNode.removeChild(child)

Elementni ota elementdan olib tashlaydi.

        
let parent = document.getElementById('parentId');
let child = document.getElementById('childId');
parent.removeChild(child);

4. parentNode.replaceChild(newChild, oldChild)

Elementni boshqa element bilan almashtiradi.

        
let parent = document.getElementById('parentId');
let newChild = document.createElement('div');
let oldChild = document.getElementById('childId');
parent.replaceChild(newChild, oldChild);

5. element.cloneNode(deep)

Elementni klonlaydi (deep true bo'lsa, barcha bola elementlar ham klonlanadi).

        
let element = document.getElementById('myId');
let clone = element.cloneNode(true);
console.log(clone);
